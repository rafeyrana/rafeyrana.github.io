Trash Detection in Water Channels using Image Segmentation

  • Tech Stack: Python, Keras, Tensorflow, OpenCV, Matplotlib, Scikit Learn, Annoy
  • Github URL: Image Segmentation Model and Image Similarity Web App

Part 1 : Segmentation of water channels to measure water contamination due to trash. The model architecture is based on a U-Net, built using Keras and TensorFlow, with a custom loss function to handle class imbalance. The model is trained and evaluated on a dataset of images, with performance metrics including accuracy, Intersection over Union (IoU), and the Dice coefficient. The trained model can be used to predict the presence and location of trash in new images of water channels.

Part 2 : Image Similarity Finder web application. The similarity is determined based on the features extracted from the images using a pre-trained ResNet50 model. The feature vectors of the images in the dataset are stored in an Annoy index for efficient similarity search. The web application is built using Streamlit which allows users to upload an image, specify the number of similar images they want to find, and initiate the similarity search. The similar images are then displayed on the web page.
